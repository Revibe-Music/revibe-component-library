### Tags Syntax

Importing the tags.
```js static
import Tags from 'revibe-component-library/Tags'

import { Tags } from 'revibe-component-library'
```

### Tags Examples
```js padded
<Tags />
<br />
<Tags color="primary" />
<br />
<Tags color="primary" tags={[ "tagA", "tagB" ]} />
```
